Programa de Asignación de Aulas

Gestión académica simplificada, clara y accesible

Descripción General

El Programa de Asignación de Aulas es un software desarrollado en Python y distribuido como ejecutable para Windows. Permite gestionar y visualizar la asignación de aulas de manera clara, sin necesidad de instalar Python ni dependencias adicionales.

Nota: El sistema debe ejecutarse cada vez que se necesite; no se instala de forma permanente ni se integra como aplicación del sistema.

Características Principales

  • Visualización clara de asignación de aulas y horarios.
  • Ejecutable independiente, sin instalación de Python.
  • Distribución local: todos los datos permanecen en el equipo del usuario.

Instrucciones de Uso

  1. Descargue el ejecutable de la versión deseada.
  2. Guarde el archivo en una carpeta propia o institucional.
  3. Ejecute el sistema haciendo doble clic sobre el archivo cada vez que lo necesite.
Importante:
  • El sistema está diseñado para ejecutarse localmente; no almacena datos en la nube.
  • No se instala como una aplicación del sistema; debe abrirse manualmente.
  • Guarde el ejecutable en una carpeta separada para evitar conflictos.
  • Si requiere una versión personalizada, solicítela mediante el repositorio.
