Programa de Asignación de Aulas

Descripción General

El Programa de Asignación de Aulas es un software desarrollado en Python y distribuido como ejecutable para Windows. Permite mostrar y gestionar la asignación de aulas de manera visual y organizada, funcionando en cualquier ordenador con Windows sin requerir la instalación de Python ni dependencias adicionales.

Nota: El sistema está diseñado para uso local y no es una aplicación instalada permanentemente; debe abrirse cada vez que se necesite. Como es un ejecutable, debe ser ejecutado manualmente y no se integra como una aplicación del sistema.

Características Principales

  • Visualización clara de la asignación de aulas y horarios.
  • Ejecutable independiente: no requiere instalación de Python.
  • Distribución local: los datos y la operación se mantienen en el equipo del usuario.

Instrucciones de Uso

  1. Descargue el ejecutable de la versión deseada.
  2. Guarde el archivo en una carpeta propia o institucional.
  3. Ejecute el sistema haciendo doble clic sobre el archivo cada vez que lo necesite.
Importante:
  • El sistema está diseñado para ejecutarse localmente; no almacena datos en la nube ni realiza sincronización remota.
  • No se instala como una aplicación del sistema; debe abrirse manualmente cada vez que se requiera consultar o gestionar las asignaciones.
  • Para evitar posibles conflictos, guarde el ejecutable en una carpeta separada y no lo modifique.
  • Si requiere una versión personalizada, comuníquese a través del repositorio para solicitar soporte.
